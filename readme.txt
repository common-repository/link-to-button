Add button style to a tag link in tinymce editor

== Description ==
Add button style to a tag link in tinymce editor like Bootstrap 4 Button: https://getbootstrap.com/docs/4.0/components/buttons/


== Installation ==

This section describes how to install the plugin and get it working.
1. Unzip the plugin contents to the `/wp-content/plugins/link-to-button/` directory
2. Activate the plugin through the \\\'Plugins\\\' menu in WordPress.
